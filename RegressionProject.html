Impact of type of transmission on mileage

Executive Summary

The objective of the below analysis is to answer the following questions using the data on fuel consumption from 1974 Motor Trend US magazine.

  • Is an automatic or manual transmission better for mileage per gallon (MPG)
  • Quantify the mileage per gallon (mpg) difference between automatic and manual transmissions

If we assume a simplistic linear relationship between the type of transmission and the mileage and ignore other variables, the linear regression model indicates that cars with manual transmission have 7.245 miles/gallon more mileage compared to cars with automatic transmission. However, further observation of the R squared (0.3598) and diagnostic regression plots confirms that transmission alone cannot explain the variance in mpg and that this simplistic model is misleading.

Analysis of the correlation matrix reveals that mpg is highly correlated to the variables - weight, number of cylinders, displacement and gross horsepower. Progressively building a set of linear models to include each of the variables and checking to see if the variables helps explain the variance (using anova), finally leads us to a satisfactory model.

As per this model we can see that controlling for weight, number of cylinders and horsepower, cars with manual transmission have 1.8 miles/gallon more mileage than cars with automatic transmission. However, the p-value of transmission coefficient indicates that we do not have sufficient evidence to reject the null hypothesis.

Hence we conclude that the there is not a significant difference in mpg between automatic and manual transmission cars controlling for weight, number of cylinders and horsepower. In other words, mileage is influenced by the other variables and the type of transmission in itself has little influence.

Data Exploration

The mean of mpg for automatic and manual transmissions are 17.14 and 24.39. A Welch two sample T-test (see Appendix: Figure1) has a low p-value and shows that the difference in the mean is significant.

However, a linear model (see Appendix: Figure1) using am as the sole feature has a low R squared (0.3598). The regression diagnostic plots also confirm that transmission alone cannot explain the variances in mpg and we'll need to account for other variables.

Feature Selection

A good starting point for identifying variables that impact mileage is to look at the correlation matrix and scatter plot of mpg vs. other variables.

The variables in decreasing order of correlation to MPG are - weight, number of cylinders, displacement and gross horsepower. These variables can be used as regressors in conjunction with transmission type.

These variables will be used in a step wise fashion to build cascading linear models. For each new model, R squared of the model, p-value of the estimated coefficients and anova will be used to assess if the variable should be used or discarded.

Model development

Including the weight variable (see Appendix: Figure2) takes up the R squared to 0.75 and the p-value of the estimate of the weight coefficient indicates it is highly significant. Given this, we infer that weight is a good variable to explain MPG and include it on our model.

Including number of cylinder factor variable takes up the R squared to 0.83 and the probability of the estimate of the new coefficient indicates they are highly significant. Given this, we infer that number of cylinder is a good variable to include in the model.

Upon repeating the same process by including displacement variable, using analysis of variance (anova) we fail to reject the null hypothesis that the new model is different in explaining the sum of squares of residuals. Based on this, we decide not to include displacement in the model.

Repeating the above process for horsepower (see Appendix: Figure3) gives a final model of R*squared of 0.86 and the diagnostic plots show that the model is acceptable.

Conclusion

Interpreting coefficients of the final model we can see that holding weight, number of cylinders and horsepower constant, cars with manual transmission have 1.8 miles/gallon more mileage than cars with automatic transmission. However, the p-value (0.20) of the coefficient is high and lacks significance.

Hence we conclude that the there is not a significant difference in mpg between automatic and manual transmission cars controlling for weight, number of cylinders and horsepower.
